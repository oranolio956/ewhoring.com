Privacy Policy

We sell information, not personal data. This document describes exactly what we collect, why, and how to erase it.

1. Data controller

Oranolio Systems LLC
185 Berry St, Suite 3200, San Francisco, CA 94107
privacy@ewhoring.com

2. Data we collect

  • Analytics: page views, scroll depth, referrer. Stored in Vercel Analytics for 14 months.
  • Checkout: email, wallet hash, IP (for fraud scoring). Stored in encrypted Supabase tables.
  • Community: Discord ID + purchase tier. Stored in Discord + Stripe webhooks.

No biometric or government ID data is collected. We do not resell or rent lists.

3. Cookies

We only set essential cookies (oranolio_session and oranolio_consent). Marketing pixels are disabled by default; you must opt in via the consent modal.

4. Data subject rights

Email privacy@ewhoring.com with the subject “DATA REQUEST” from the same email you used at checkout to:

  • Request a copy of your data
  • Request deletion (processed within 72h)
  • Request payment history redaction

5. Security

  • All dashboards enforce hardware security keys (FIDO2).
  • Data at rest is encrypted with AES-256 and rotated quarterly.
  • Access logs are retained for 90 days for legal holds.

6. Changes

We ship updates aggressively. When anything privacy-related changes, we log it in the public changelog and update this page.
